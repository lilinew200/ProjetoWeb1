HTML

Com esse curso de HTML, você vai aprender a criar uma página web, utilizando e editando códigos em HTML. Esse curso de HTML foi desenvolvido para o ensino da criação e edição de páginas WEB. Atualmente, há uma grande variedade de programas e editores HTML, que facilitam a criação de site e não exigem do usuário muito conhecimento sobre o assunto, como por exemplo, o famoso dreamweaver da Adobe. Porém, com esse curso de HTML, não iremos falar sobre esse tipo de programa ou recurso para criação e edição de códigos HTML. O propósito desse curso grátis é que qualquer pessoa aprenda a criar páginas, condificando os comandos HTML, utilizando apenas o bloco de notas. Para quem é indicado esse curso grátis de HTML:

  • Qualquer pessoa que deseja aprender HTML
  • Desenvolvedores que já trabalham com HTML
  • Professores e instrutores que desejam usar esse Curso Grátis de HTML como material didático
  • Não há pré-requisitos para fazer esse curso de HTML, você deverá apenas ter o conhecimento básico em informática e um editor de texto de sua preferência, como por exemplo o Bloco de Notas.
